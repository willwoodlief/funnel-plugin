== Description ==

Install by

1) unzipping and putting into wp-content/plugins directory
2) add in the private_envs.php file
3) add pear Mail/mimeDecode @ http://pear.php.net/package/Mail_mimeDecode/
2) go into wordpress and clicking install for the plugin
